No download or environment variable is needed after cloning. The notebooks
prefer this directory automatically. `AVITO_DATA_DIR` remains available when a
different copy of the dataset must be used.

Before publishing a repository publicly, confirm that the competition dataset's
redistribution terms permit it. A private repository is the safe default.
